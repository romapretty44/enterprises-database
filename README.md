## 🚀 Шаги для запуска

### 1. Включить Firestore в Firebase Console

1. Откройте [Firebase Console](https://console.firebase.google.com/)
2. Выберите проект **enterprises-database**
3. В боковом меню: **Firestore Database** → **Create database**
4. Выберите **"Start in test mode"** (для начала)
5. Регион: **europe-west** или ближайший
6. Нажмите **Enable**

### 2. Настроить правила безопасности

Во вкладке **Rules** в Firestore вставьте:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true; // Открыто для всех (временно!)
    }
  }
}
```

⚠️ **Важно:** Это открытые правила для тестирования. Позже настроим безопасность!

Нажмите **Publish**

### 3. Загрузить обновлённые файлы на GitHub

1. Откройте [ваш репозиторий](https://github.com/romapretty44/enterprises-database)
2. Нажмите **Add file** → **Upload files**
3. Загрузите эти файлы (заменят старые):
   - `index.html`
   - `app.js`
   - `firebase-init.js` ⬅️ **НОВЫЙ ФАЙЛ**
   - `styles.css` (если изменили)
4. Commit message: `Переход на Firestore - облачная БД`
5. Нажмите **Commit changes**

### 4. Подождать 1-2 минуты

GitHub Pages автоматически обновится. Затем откройте:  
👉 https://romapretty44.github.io/enterprises-database/
