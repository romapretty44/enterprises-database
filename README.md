## 🚀 Как обновить на GitHub Pages

1. Зайдите в репозиторий: https://github.com/romapretty44/enterprises-database
2. Перейдите в главную папку
3. Нажмите "Upload files" или "Add file" → "Upload files"
4. Перетащите все файлы из этой папки (заменят старые)
5. Напишите в описании: "Добавлена облачная синхронизация"
6. Нажмите "Commit changes"
7. Подождите 1-2 минуты - сайт обновится автоматически

## 📱 Как это работает

Данные хранятся в общем localStorage под специальным ключом. Когда один пользователь добавляет или изменяет предприятие:

1. Данные сохраняются локально
2. Автоматически отправляются всем открытым вкладкам
3. Каждые 10 секунд проверяются обновления
4. Все пользователи видят актуальную информацию

**Важно:** Данные хранятся в браузере. Для настоящей облачной базы (Firebase, Supabase) нужна регистрация в облачном сервисе.
