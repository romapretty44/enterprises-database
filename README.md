## 🗄️ Структура Firestore

### Коллекция `enterprises`
Основная коллекция предприятий с полями:
- `name` - название предприятия
- `info` - описание деятельности
- `industries` - массив отраслей
- `categories` - объект с булевыми флагами категорий
- `regionalSupportDesc` - описание региональной поддержки
- `federalSupportDesc` - описание федеральной поддержки
- `contacts` - массив контактов
- `files` - массив загруженных файлов
- `createdAt` - дата создания
- `updatedAt` - дата обновления

### Коллекция `trash`
Корзина удалённых предприятий:
- Все поля из `enterprises`
- `deletedAt` - дата удаления
- `originalId` - исходный ID документа

### Коллекция `industries`
Динамический список отраслей:
- `name` - название отрасли

## 📊 Мониторинг

### Firebase Console
Проверяйте регулярно:
- Использование Firestore (чтения/записи)
- Объём Storage
- Трафик Hosting

### Квоты бесплатного плана
- Firestore: 50K чтений/день, 20K записей/день
- Storage: 5 GB хранения, 1 GB загрузки/день
- Hosting: 10 GB хранения, 360 MB/день трафика
