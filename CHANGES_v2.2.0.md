# Изменения v2.2.0 - Динамические категории

## 📅 Дата: 26.03.2026

## ✨ **ГЛАВНАЯ ФИЧА: Динамические категории**

Теперь категории работают **точно так же, как отрасли** - динамически создаются через UI!

### 🎯 Что реализовано:

#### 1. **Новая коллекция Firestore:**
```javascript
categories: [
  {
    id: "auto-generated-id",
    name: "ОПК",
    hasDescription: false
  },
  {
    id: "auto-generated-id",
    name: "Получатель поддержки региональной",
    hasDescription: true
  }
]
```

#### 2. **UI для добавления категорий:**
- ➕ Кнопка "Добавить категорию" в форме редактирования
- 🔲 Модальное окно с полями:
  - Название категории
  - Чекбокс "Категория имеет поле для описания"
- ✅ Валидация (название не может быть пустым или дублироваться)

#### 3. **Новая структура данных предприятия:**

**Было (старая система):**
```javascript
{
  categories: {
    opk: boolean,
    categorized: boolean,
    ...
  },
  regionalSupportDesc: string,
  federalSupportDesc: string
}
```

**Стало (динамическая система):**
```javascript
{
  categories: {
    "categoryId1": true,
    "categoryId2": true,
    ...
  },
  categoriesDescriptions: {
    "categoryId1": "Описание поддержки...",
    "categoryId2": "Другое описание..."
  }
}
```

#### 4. **Автоматическое управление textarea:**
- При выборе категории с описанием → textarea появляется
- При снятии галочки → textarea скрывается
- При редактировании → значения загружаются автоматически

## ✅ **СОВМЕСТИМОСТЬ:**

### Миграция данных:

Старые предприятия с boolean категориями **НЕ будут работать** автоматически.

**Решение:**
1. Создать скрипт миграции данных
2. Преобразовать старые `categories.opk` в новые ID категорий
3. Перенести `regionalSupportDesc` → `categoriesDescriptions[categoryId]`

**Примечание:** Если в базе нет старых данных - миграция не требуется.

## 📝 **КАК ИСПОЛЬЗОВАТЬ:**

### Добавить категорию:

1. Открыть форму добавления/редактирования предприятия
2. Прокрутить до раздела "📂 Дополнительные категории"
3. Нажать **"➕ Добавить категорию"**
4. Ввести название
5. Отметить чекбокс, если нужно поле описания
6. Нажать **"Добавить"**

### Использовать категорию:

1. Отметить нужные категории
2. Если категория имеет описание - заполнить textarea
3. Сохранить предприятие

### Фильтровать:

1. В левой панели выбрать категории
2. Предприятия фильтруются автоматически
