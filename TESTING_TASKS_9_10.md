# Тестирование задач 9-10

## 🧪 Задача 9: Руководители предприятия

### Шаги тестирования:

1. **Добавление руководителей:**
   - Откройте модальное окно добавления предприятия
   - Найдите секции:
     - "👔 Руководитель предприятия"
     - "🤝 Помощник руководителя предприятия"
   - Заполните поля (необязательно):
     - ФИО
     - Должность
     - Контактный телефон
   - Сохраните предприятие

2. **Просмотр:**
   - Нажмите "👁️ Просмотр" на карточке предприятия
   - Проверьте отображение секций:
     - "👔 Руководитель предприятия"
     - "🤝 Помощник руководителя предприятия"
   - Данные должны показываться в отдельных карточках

3. **Редактирование:**
   - Нажмите "✏️ Редактировать"
   - Проверьте, что поля руководителей заполнены
   - Измените данные
   - Сохраните и проверьте просмотр

### ✅ Ожидаемый результат:
- Поля сохраняются в Firestore
- Отображаются в просмотре
- Заполняются при редактировании
- Можно оставить пустыми (необязательные)

## 🧪 Задача 10: Раздел "Производство"

### Шаги тестирования:

#### A. Добавление видов продукции

1. **Первое предприятие:**
   - Откройте модальное окно добавления
   - Найдите секцию "🏭 Производство (виды продукции)"
   - Нажмите "➕ Добавить вид продукции"
   - Введите вид продукции (например, "Металлоконструкции")
   - Нажмите "➕ Добавить вид продукции" ещё раз
   - Введите другой вид (например, "Запасные части")
   - Сохраните предприятие

2. **Второе предприятие с автокомплитом:**
   - Откройте модальное окно добавления
   - Нажмите "➕ Добавить вид продукции"
   - Начните вводить "Мет..." - должен появиться автокомплит
   - Выберите существующий вид из списка
   - Добавьте новый вид продукции
   - Сохраните

3. **Удаление вида:**
   - При добавлении продукции нажмите "❌" рядом с полем
   - Поле должно удалиться

#### B. Фильтрация по видам продукции

1. **Появление фильтров:**
   - После добавления хотя бы одного предприятия с продукцией
   - На главной странице должна появиться секция "Виды производимой продукции:"
   - Чекбоксы создаются динамически

2. **Фильтрация:**
   - Выберите один или несколько видов продукции
   - Должны показаться предприятия, у которых есть хотя бы один из выбранных видов
   - Логика: OR (любой из выбранных)

3. **Просмотр:**
   - Откройте предприятие на просмотр
   - Найдите секцию "🏭 Производство"
   - Виды продукции должны отображаться в виде зелёных тегов

4. **Редактирование:**
   - Откройте предприятие на редактирование
   - Все виды продукции должны быть в списке
   - Можно добавить новые
   - Можно удалить существующие

### ✅ Ожидаемый результат:
- Виды продукции сохраняются в массиве `productionTypes`
- Новые виды добавляются в коллекцию `products`
- Автокомплит работает (datalist)
- Фильтры на главной появляются динамически
- Фильтрация работает (OR логика)
- Отображение в виде тегов в просмотре

     }
     assistant: {
       fullName: "...",
       position: "...",
       phone: "..."

2. **Коллекция `products` (новая!):**
   - Должна автоматически создаться
   - Каждый документ содержит:
     ```
     name: "Название вида продукции"
     ```

## 🐛 Возможные проблемы

### Проблема: Фильтры по продукции не появляются
**Решение:**
- Добавьте хотя бы одно предприятие с видом продукции
- Обновите страницу
- Проверьте консоль браузера на ошибки

### Проблема: Автокомплит не работает
**Решение:**
- Убедитесь, что уже добавлены виды продукции
- Проверьте, что коллекция `products` создана
- Начните вводить несколько букв

### Проблема: Данные руководителей не сохраняются
**Решение:**
- Проверьте консоль браузера
- Убедитесь, что поля заполнены
- Проверьте Firestore Rules
