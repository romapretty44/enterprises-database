# 📧 Обновлённый формат выгрузки почт

## Изменения

Изменён формат выгрузки почт для рассылки с простого списка через запятую на **нумерованный список с названиями компаний**.

## Старый формат ❌

```
email1@mail.ru, email2@mail.ru, email3@mail.ru, email4@mail.ru
```

**Проблемы:**
- Не видно, какой email к какой компании относится
- Сложно проверить полноту списка
- Неудобно для дальнейшей работы

## Новый формат ✅

```
1. АО "Металлургический завод" - info@metal.ru
2. ООО "Судоремонтный комплекс" - office@shiprepair.ru, pr@shiprepair.ru
3. ИП Иванов И.И. - ivan@business.ru
4. ПАО "Машиностроительный завод" - contact@machinery.ru, sales@machinery.ru, hr@machinery.ru
```

**Преимущества:**
- ✅ Видно название каждой компании
- ✅ Все почты компании сгруппированы
- ✅ Нумерация для удобного подсчёта
- ✅ Структурированный и читаемый формат
- ✅ Легко найти конкретное предприятие

## Структура формата

```
<номер>. <Название предприятия> - <email1>, <email2>, <email3>
```

**Компоненты:**
- `<номер>` - порядковый номер предприятия (1, 2, 3, ...)
- `<Название предприятия>` - полное название из поля `name`
- ` - ` - разделитель (пробел-тире-пробел)
- `<email1>, <email2>` - все почты предприятия через запятую с пробелом

## Реализация

### JavaScript код:
```javascript
// Формируем нумерованный список с названиями компаний
const mailingList = [];
let totalEmailsCount = 0;
let index = 1;

filtered.forEach(ent => {
    if (ent.mailingEmails && ent.mailingEmails.length > 0) {
        const companyName = ent.name;
        const emails = ent.mailingEmails.join(', ');
        mailingList.push(`${index}. ${companyName} - ${emails}`);
        totalEmailsCount += ent.mailingEmails.length;
        index++;
    }
});

// Формируем текст с переносами строк
const resultText = mailingList.join('\n');
```

### Счётчик обновлён:
**Было:**
```
Всего почт: 45
```

**Стало:**
```
Всего предприятий: 15 | Всего почт: 45
```

Показывает:
- Количество предприятий с почтами
- Общее количество email адресов

## HTML отображение

### Textarea стили:
```css
textarea {
    font-family: 'Courier New', monospace;  /* Моноширинный шрифт */
    font-size: 13px;                        /* Читаемый размер */
    line-height: 1.6;                       /* Хорошая высота строки */
    white-space: pre;                       /* Сохранение переносов */
    min-height: 300px;                      /* Увеличена высота */

```

**Зачем моноширинный шрифт?**
- Все цифры номеров выровнены
- Легче читать структуру
- Профессиональный вид

## Примеры использования

### Пример 1: Одна почта на компанию
```
1. АО "Компания А" - contact@company-a.ru
2. ООО "Компания Б" - info@company-b.ru
3. ИП Петров - petrov@business.ru
```

### Пример 2: Несколько почт
```
1. АО "Крупное предприятие" - main@big.ru, sales@big.ru, hr@big.ru
2. ООО "Малое предприятие" - office@small.ru
```

### Пример 3: После фильтрации
**Фильтр:** Отрасль "Металлургия" + Категория "ОПК"

**Результат:**
```
1. ПАО "Металлургический комбинат №1" - info@metal1.ru
2. АО "Специальная металлургия" - contact@specmetal.ru, zakaz@specmetal.ru
3. ООО "Металлопрокат ОПК" - office@metalopk.ru
```

Счётчик: `Всего предприятий: 3 | Всего почт: 4`

## Копирование

При нажатии кнопки "📋 Копировать весь список":
1. Выделяется весь текст из textarea
2. Копируется в буфер обмена
3. Кнопка меняется на "✅ Скопировано!" на 2 секунды
4. Можно вставить в:
   - Документ Word/Excel
   - Email клиент
   - Таблицу Google Sheets
   - Текстовый файл

## Обработка edge cases

### Нет почт у предприятия
Предприятие **не попадает** в список, если `mailingEmails` пустой или null.

### Одно предприятие с 10 почтами
```
1. ООО "Большая компания" - mail1@big.ru, mail2@big.ru, mail3@big.ru, mail4@big.ru, mail5@big.ru, mail6@big.ru, mail7@big.ru, mail8@big.ru, mail9@big.ru, mail10@big.ru
```
Все почты отображаются в одной строке.

### Название с кавычками
Кавычки в названии компании сохраняются как есть:
```
1. АО "Завод "Прогресс"" - info@progress.ru
```

### Спецсимволы в названии
```
1. ООО "Металл&Сплавы" - contact@metal-alloys.ru
```
Символы `&`, `<`, `>` отображаются корректно (textarea не интерпретирует HTML).

## Интеграция с Excel/Google Sheets

Скопированный список легко вставляется в таблицу:

**В Excel/Sheets вставка создаёт:**

| Строка | Содержимое |
|--------|------------|
| 1 | 1. АО "Компания А" - contact@company-a.ru |
| 2 | 2. ООО "Компания Б" - info@company-b.ru |
| 3 | 3. ИП Петров - petrov@business.ru |

Можно далее разделить по столбцам:
- Колонка A: Номер
- Колонка B: Название
- Колонка C: Email(ы)

**Формула разделения в Excel:**
```excel
=TRIM(LEFT(A1, FIND("-", A1)-1))    // Название
=TRIM(RIGHT(A1, LEN(A1)-FIND("-", A1)))  // Почты
```
