## [2.0.0] - 2026-03-26

### ✨ Новые функции

#### 1. Динамические фильтры (отрасли)
- Возможность создавать новые отрасли прямо в интерфейсе
- Кнопка "➕ Добавить отрасль" в разделе фильтров
- Новые отрасли автоматически сохраняются в Firestore коллекцию `industries`
- Мгновенное обновление списков фильтров и форм

#### 2. Загрузка файлов
- Интеграция с Firebase Storage
- Поддержка любых типов файлов (документы, изображения, PDF и т.д.)
- Множественная загрузка файлов
- Просмотр и скачивание файлов из карточки предприятия
- Удаление файлов при редактировании

#### 3. Экспорт базы данных
- Кнопка "📊 Экспорт в Excel" в шапке приложения
- Экспорт в CSV формате (открывается в Excel)
- Включает все данные: название, информация, отрасли, категории, поддержка, контакты
- Автоматическое именование файла с датой

#### 4. Корзина удалённых
- Мягкое удаление предприятий
- Отдельная коллекция `trash` в Firestore
- Кнопка "🗑️ Корзина" для просмотра удалённых
- Функции:
  - ♻️ Восстановление предприятия
  - 🗑️ Окончательное удаление
- Сохранение даты удаления

#### 5. Компактные карточки
- На главной странице показываются только название и отрасли
- Описание скрыто для компактности
- Кнопка "👁️ Просмотр" открывает модальное окно с полной информацией
- Улучшенная читаемость и производительность

#### 6. Фильтрация AND
- Изменена логика фильтрации по отраслям
- Теперь показываются предприятия, содержащие ВСЕ выбранные отрасли
- Более точная фильтрация для сложных запросов
- Подсказка в интерфейсе "(AND - все выбранные)"

#### 7. Дополнительные категории
- Новые чекбоксы в форме предприятия:
  - ✅ Категорируется
  - ✅ Участник производительности труда
  - ✅ Участник профессионалитета
  - ✅ Получатель поддержки региональной (+ текстовое поле описания)
  - ✅ Получатель поддержки федеральной (+ текстовое поле описания)
- Отдельная секция фильтрации по категориям
- Отображение категорий в модальном окне просмотра

#### 8. Раздел контактов
- Блок "👥 Контакты" в форме предприятия
- Поля для каждого контакта:
  - ФИО
  - Должность
  - Рабочий телефон
  - Почта
  - Мобильный телефон
- Кнопка "➕ Добавить контакт" для неограниченного количества контактов
- Кнопка "❌" для удаления контактов
- Красивое отображение контактов в карточках

### 🎨 Улучшения дизайна

- Обновлены стили для модальных окон (маленький, средний, большой)
- Новые кнопки в header: Экспорт и Корзина
- Улучшенная типографика для форм
- Адаптивные секции фильтров
- Стили для контакт-форм
- Стили для списка файлов

### 🔧 Технические улучшения

- Добавлен Firebase Storage SDK
- Новая структура Firestore коллекций
- Улучшенная обработка ошибок
- Валидация данных перед сохранением
- Оптимизация рендеринга карточек
- ESC функция для безопасного вывода HTML

### 📦 Новые файлы

- `storage.rules` - правила безопасности Firebase Storage
- `TESTING.md` - руководство по тестированию
- `CHANGELOG.md` - история изменений

### 🔄 Обновлённые файлы

- `index.html` - новая разметка с модальными окнами и формами
- `app.js` - полная переработка логики приложения
- `styles.css` - новые стили для всех компонентов
