# 📧 Функция "Почты для рассылок"

## Описание

Добавлена полноценная система управления email адресами для массовых рассылок с фильтрацией по отраслям и категориям.

## Функциональность

### 1. Управление почтами в карточке предприятия

**Расположение:** Форма добавления/редактирования предприятия  
**Секция:** "📧 Почты для рассылок" (после контактов)

**Возможности:**
- ✅ Добавление нескольких email адресов
- ✅ Валидация формата email
- ✅ Удаление каждой почты отдельно
- ✅ Сохранение в Firestore

**HTML:**
```html
<div class="form-section">
    <label>📧 Почты для рассылок</label>
    <div id="mailingEmailsList"></div>
    <button type="button" id="addMailingEmailBtn">➕ Добавить почту</button>
</div>
```

**Валидация email:**
```javascript
function isValidEmail(email) {
    const re = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return re.test(email);
}
```

При попытке сохранить некорректный email выводится alert:
```
Некорректный email: invalid.email
```

### 2. Кнопка "📧 Выгрузка рассылки"

**Расположение:** Header, рядом с "📊 Экспорт в Excel"

**Действие:** Открывает модальное окно с выбором фильтров

---

### 3. Модальное окно выбора фильтров

**Содержит:**
- 🏭 Мультиселект отраслей (чекбоксы)
- 📂 Мультиселект категорий (чекбоксы)
- 💡 Подсказка: "Если ничего не выбрано, будут выгружены почты всех предприятий"
- Кнопка "📧 Выгрузить почты"

**Логика фильтрации:**
```javascript
const matchesIndustries = selectedIndustries.length === 0 || 
    selectedIndustries.some(ind => ent.industries && ent.industries.includes(ind));

const matchesCategories = selectedCategories.length === 0 || 
    selectedCategories.some(cat => ent.categories && ent.categories[cat]);

return matchesIndustries && matchesCategories;
```

- Если **ничего не выбрано** → все предприятия
- Если выбраны **только отрасли** → фильтр по отраслям (OR)
- Если выбраны **только категории** → фильтр по категориям (OR)
- Если выбраны **и отрасли, и категории** → фильтр по обоим (AND между группами, OR внутри группы)

**Отображает:**

**Функционал:**
- Textarea с результатом (readonly)
- Кнопка "📋 Копировать все почты"

  - Выделяется весь текст
  - Копируется в буфер обмена
  - Кнопка меняется на "✅ Скопировано!" на 2 секунды

**Копирование:**
```javascript
const textarea = document.getElementById('mailingEmailsText');
textarea.select();
document.execCommand('copy');

// Feedback
btn.textContent = '✅ Скопировано!';
setTimeout(() => {
    btn.textContent = '📋 Копировать все почты';
}, 2000);
```

### В форме редактирования (editEnterprise)

Автоматически загружаются существующие почты:
```javascript
if (ent.mailingEmails) {
    ent.mailingEmails.forEach(email => {
        const emailHTML = `
            <div data-mailing-email-id="${emailId}">
                <input type="email" class="mailing-email-input" value="${email}">
                <button onclick="removeMailingEmail(${emailId})">❌</button>
            </div>
        `;
        mailingEmailsList.insertAdjacentHTML('beforeend', emailHTML);
    });
}
```

## Удаление дубликатов

При выгрузке автоматически убираются дубликаты:
```javascript
const allEmails = [];
filtered.forEach(ent => {
    if (ent.mailingEmails && ent.mailingEmails.length > 0) {
        ent.mailingEmails.forEach(email => {
            if (!allEmails.includes(email)) {  // ✅ Проверка на дубликат
                allEmails.push(email);
            }
        });
    }
});
```

## UI/UX особенности

1. **Валидация email в реальном времени**
   - При сохранении проверяется формат каждого email
   - Некорректные email блокируют сохранение

2. **Визуальная обратная связь**
   - "✅ Скопировано!" появляется на 2 секунды
   - Счётчик "Всего почт: N" показывает количество

3. **Удобство использования**
   - Textarea с автоматическим выделением при клике
   - Возможность ручного редактирования перед копированием
   - Вертикальная прокрутка для больших списков

4. **Очистка при создании нового предприятия**
   - Поле mailingEmails сбрасывается при открытии формы добавления
