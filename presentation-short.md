# 🏭 База предприятий
### Система управления данными предприятий

Современное веб-приложение для учета и управления информацией о предприятиях с интеграцией ЕГРЮЛ

---

# 🎯 Основные возможности

## Ключевой функционал

- **📊 Управление предприятиями** - добавление, редактирование, удаление
- **🔍 Умная фильтрация** - по отраслям, категориям, видам продукции
- **🏢 Интеграция с ЕГРЮЛ** - автозагрузка данных через DaData API
- **📤 Экспорт в CSV** - полная выгрузка данных для Excel
- **🗑️ Корзина** - безопасное удаление с восстановлением
- **☁️ Облачное хранилище** - Firebase Firestore

---

# 📝 Работа с предприятиями

## Полная информация о каждом предприятии

**Основные данные:**
- Название и описание
- Отрасли (множественный выбор)
- Дополнительные категории с описаниями
- Виды производимой продукции

**Контакты:**
- Руководители (директор, помощник)
- Контактные лица с должностями
- Email-адреса для рассылок

**Юридическая информация:**
- ИНН, ОГРН
- ОКВЭД с расшифровкой
- Юридический адрес
- Дата регистрации

---

# 🏢 Интеграция с ЕГРЮЛ

## Автоматическая загрузка по ИНН через DaData API

**Что загружается автоматически:**
- ✅ Полное наименование организации
- ✅ ОГРН
- ✅ ОКВЭД (основной + дополнительные с расшифровкой)
- ✅ Юридический адрес
- ✅ Дата регистрации компании
- ✅ Руководитель (ФИО и должность)

**Преимущества:**
- ⚡ Быстрое заполнение - достаточно ввести ИНН
- ✏️ Все данные редактируемые
- 🔍 Проверка на дубликат ИНН при сохранении

# 📤 Экспорт и выгрузка данных

## CSV экспорт одним кликом

**Что выгружается:**

📋 **Основная информация**
- Название предприятия
- Описание деятельности
- Список отраслей
- Категории с описаниями

🏢 **Юридические данные**
- ИНН, ОГРН
- ОКВЭД с полной расшифровкой

- Контактные лица с должностями и телефонами

# 🗑️ Безопасное удаление

## Корзина с возможностью восстановления

**Как работает:**
1. При удалении предприятие перемещается в корзину
2. Сохраняются все данные + время удаления
3. Можно восстановить одним кликом
4. Окончательное удаление с подтверждением

**Утилиты администрирования:**
- 🗑️ **clean-trash.html** - полная очистка корзины
- 🏷️ **manage-categories.html** - управление категориями
- 🔄 **reload-site.html** - перезагрузка сайта

# 🛠️ Технологии и архитектура

## Современный технологический стек

**Frontend:**
- HTML5/CSS3/JavaScript (ES6+)
- Firebase SDK для работы с Firestore
- Шрифт Inter (профессиональный дизайн)

**Backend (Serverless):**
- Firebase Firestore - облачная NoSQL база
- Firebase Hosting - быстрая доставка контента
- GitHub Pages - публикация

**Интеграции:**
- DaData API - данные ЕГРЮЛ
- GitHub API - автоматизация деплоя

**Преимущества архитектуры:**
✅ Без сервера (serverless)  
✅ Автоматические бэкапы  
✅ Масштабируемость  
✅ Real-time синхронизация
