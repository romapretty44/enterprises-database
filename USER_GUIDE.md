# Руководство пользователя - База предприятий

## 🔐 Вход в систему

1. Откройте приложение в браузере
2. Введите пароль: `444455555`
3. Нажмите "Войти"

## 📝 Добавление предприятия

1. Нажмите кнопку **"➕ Добавить предприятие"** в верхней части экрана
2. Заполните обязательные поля:
   - **Название предприятия** *
   - **Отрасли** * (выберите хотя бы одну)
3. Заполните дополнительные поля по желанию:
   - **Информация о предприятии** - описание деятельности
   - **Дополнительные категории** - статус предприятия
   - **Контакты** - контактные лица
   - **Файлы** - документы и изображения
4. Нажмите **"💾 Сохранить"**

## 👀 Просмотр предприятия

На главной странице показываются компактные карточки предприятий с названием и отраслями.

**Чтобы увидеть полную информацию:**
1. Нажмите кнопку **"👁️ Просмотр"** на карточке
2. Откроется модальное окно со всеми деталями:
   - Полное описание
   - Все отрасли
   - Категории и статусы
   - Контакты
   - Прикреплённые файлы

## ✏️ Редактирование предприятия

1. Нажмите **"✏️ Редактировать"** на карточке
2. Измените нужные поля
3. Нажмите **"💾 Сохранить"**

## 🗑️ Удаление предприятия

### Мягкое удаление (в корзину)
1. Нажмите **"🗑️ Удалить"** на карточке
2. Подтвердите удаление
3. Предприятие переместится в корзину

### Восстановление из корзины
1. Нажмите **"🗑️ Корзина"** в верхней части экрана
2. Найдите нужное предприятие
3. Нажмите **"♻️ Восстановить"**

### Окончательное удаление
1. Откройте корзину
2. Нажмите **"🗑️ Удалить навсегда"**
3. Подтвердите действие (это необратимо!)

## 🔍 Поиск и фильтрация

### Поиск
Введите текст в поле **"🔍 Поиск"**. Поиск работает по:
- Названию предприятия
- Описанию

### Фильтрация по отраслям
1. В разделе **"Фильтр по отраслям"** выберите нужные отрасли
2. Будут показаны только предприятия, у которых есть **ВСЕ** выбранные отрасли (логика AND)

**Пример:**
- Выбраны: "Промышленность" + "ОПК"
- Показаны: только предприятия, которые одновременно относятся к обеим отраслям

### Фильтрация по категориям
В разделе **"Дополнительные категории"** выберите:
- Категорируется
- Участник производительности труда
- Участник профессионалитета
- Получатель поддержки региональной
- Получатель поддержки федеральной

## ➕ Добавление новой отрасли

Если нужной отрасли нет в списке:
1. В разделе фильтров нажмите **"➕ Добавить отрасль"**
2. Введите название новой отрасли
3. Нажмите **"Добавить"**
4. Новая отрасль появится во всех списках

## 👥 Работа с контактами

### Добавление контакта
1. В форме предприятия найдите раздел **"👥 Контакты"**
2. Нажмите **"➕ Добавить контакт"**
3. Заполните поля:
   - ФИО
   - Должность
   - Рабочий телефон
   - Почта
   - Мобильный телефон
4. Добавьте столько контактов, сколько нужно

### Удаление контакта
Нажмите кнопку **"❌"** справа от контактной формы

## 📎 Работа с файлами

### Загрузка файлов
1. В форме предприятия найдите раздел **"📎 Файлы"**
2. Нажмите **"Выбрать файлы"**
3. Выберите один или несколько файлов (любой тип)
4. Файлы загрузятся при сохранении предприятия

### Просмотр файлов
1. Откройте предприятие на просмотр
2. В разделе **"📎 Файлы"** кликните на название файла
3. Файл откроется в новой вкладке или скачается

### Удаление файлов
1. Отредактируйте предприятие
2. В списке файлов нажмите **"❌"** рядом с ненужным файлом
3. Сохраните изменения

## 📊 Экспорт базы данных

1. Нажмите **"📊 Экспорт в Excel"** в верхней части экрана
2. Скачается CSV-файл с названием `enterprises_ГГГГ-ММ-ДД.csv`
3. Откройте файл в Excel:
   - Данные → Из текста/CSV
   - Разделитель: точка с запятой (;)

**Экспортируются:**
- Все предприятия
- Вся информация (название, описание, отрасли, категории, контакты)

## 🚪 Выход

Нажмите **"🚪 Выйти"** в верхней части экрана

## 💡 Полезные советы

### Категории предприятий
- **Категорируется** - предприятие прошло категорирование
- **Участник производительности труда** - участвует в программе повышения производительности
- **Участник профессионалитета** - участвует в образовательной программе
- **Поддержка региональная/федеральная** - получает государственную поддержку (укажите детали в текстовых полях)

### Real-time синхронизация
- Все изменения видны сразу всем пользователям
- Не нужно обновлять страницу
- Работает через облачную базу данных Firestore

### Безопасность данных
- Все данные хранятся в облаке Firebase
- Автоматическое резервное копирование
- Защита паролем

## ❓ Частые вопросы

**Q: Можно ли работать с нескольких устройств одновременно?**  
A: Да! Все изменения синхронизируются в реальном времени.

**Q: Какой максимальный размер файла?**  
A: До 50 МБ для одного файла.

**Q: Что делать, если случайно удалил предприятие?**  
A: Откройте корзину и восстановите предприятие. Из корзины можно восстановить в течение любого времени.

**Q: Можно ли добавить своё поле в карточку?**  
A: Сейчас нет, но это планируется в будущих версиях.

**Q: Как изменить пароль?**  
A: Обратитесь к администратору системы.

## 🆘 Проблемы?

Если что-то не работает:
1. Обновите страницу (F5 или Ctrl+R)
2. Очистите кэш браузера
3. Проверьте интернет-соединение
4. Откройте консоль браузера (F12) и проверьте ошибки
5. Обратитесь к администратору
