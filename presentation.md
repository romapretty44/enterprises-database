# 🏭 База предприятий
### Система управления данными предприятий

Современное веб-приложение для учета и управления информацией о предприятиях с интеграцией ЕГРЮЛ

---

# 🎯 Основные возможности

- **📊 Управление предприятиями** - добавление, редактирование, удаление
- **🔍 Фильтрация и поиск** - по отраслям, категориям, видам продукции
- **🏢 Интеграция с ЕГРЮЛ** - автозагрузка юридических данных через DaData API
- **📤 Экспорт данных** - выгрузка в CSV формат
- **🗑️ Корзина** - безопасное удаление с возможностью восстановления
- **☁️ Облачное хранилище** - Firebase Firestore для надежности данных

---

# 📝 Работа с предприятиями

## Добавление предприятия
- Название и описание
- Выбор отраслей (множественный выбор)
- Дополнительные категории с описаниями
- Виды производимой продукции
- Руководители (директор, помощник)
- Контактные лица
- Email-адреса для рассылок

## Редактирование
- Полное редактирование всех полей
- Проверка на дубликат ИНН
- Сохранение истории изменений

---

# 🔍 Фильтрация и поиск

## Умные фильтры
- **По отраслям** - Промышленность, Судоремонт, Металлургия и др.
- **По категориям** - ОПК, Категорируется, Участник программ
- **По видам продукции** - динамический список из реальных данных

## Особенности
- ✅ Множественный выбор фильтров
- ✅ Фильтры обновляются в реальном времени
- ✅ Виды продукции берутся только из активных предприятий
- ✅ Автоматическое удаление неиспользуемых фильтров

---

# 🏢 Интеграция с ЕГРЮЛ

## Автоматическая загрузка данных
**Через DaData API по ИНН:**

### Загружаются автоматически:
- ✅ Полное наименование организации
- ✅ ОГРН
- ✅ ОКВЭД (основной + дополнительные с расшифровкой)
- ✅ Юридический адрес
- ✅ Дата регистрации (из ogrn_date)
- ✅ Руководитель (ФИО и должность)

### Редактирование
- Все загруженные данные можно отредактировать вручную
- Проверка на дубликат ИНН при сохранении

---

# 📤 Экспорт данных

## CSV экспорт
**Одним кликом выгружается полная информация:**

- Название предприятия
- Описание
- Отрасли
- Категории с описаниями
- ИНН, ОГРН
- ОКВЭД с расшифровкой
- Юридический адрес
- Контакты
- Email-адреса для рассылок

## Формат
- Разделитель: точка с запятой (;)
- Кодировка: UTF-8
- Совместимость: Excel, Google Sheets

---

# 🗑️ Корзина и восстановление

## Безопасное удаление
- При удалении предприятие перемещается в корзину
- Сохраняются все данные + метка времени удаления
- Возможность восстановления одним кликом

## Окончательное удаление
- Кнопка "Удалить навсегда" в корзине
- Запрос подтверждения
- Документ удаляется из Firestore навсегда

## Утилиты
- **clean-trash.html** - полная очистка корзины
- **manage-categories.html** - управление категориями
- **reload-site.html** - перезагрузка GitHub Pages
